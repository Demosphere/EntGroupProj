Service Description

The Loquaciously-Worded Login Service is a RESTful API. Use your browser's address bar to pass parameters to manage users for your web application. There are four management functions which you can perform: log a user into your application, subscribe a user, update user information, and delete a user.

To log a user into your application, you will need to pass in three parameters: your site key, the username, and the user's password. Use the following url to log a user into your application: 

rest/authenticator/login/{site-key}/{username}/{password}

example: rest/authenticator/login/87d3e948-efd0-40e2-af79-2d532c390d09/Groot/password1

To subscribe a new user, you will need to pass in four parameters: your site key, the username, the user's password, and the user's role. Use the following url to subscribe a user: 

/auth-resource/subscribe{site-key}/{username}/{password}/
        {role}

example: rest/authenticator/subscribe/87d3e948-efd0-40e2-af79-2d532c390d09/Groot/password1/admin

To unsubscribe an existing user, you will need to pass in two parameters: your site key and the username. Use the following url to unsubscribe a user: 

/auth-resource/unsubscribe{site-key}/{username}

example: rest/authenticator/unsubscribe/87d3e948-efd0-40e2-af79-2d532c390d09/Groot

To update a user's information, you will need to pass in four parameters: your site key, the username, the user's new password (if applicable), the user's new role (if applicable). Use the following url to update information for a user: 

/auth-resource/update{site-key}/{username}/{password}/{role}

example: rest/authenticator/update/87d3e948-efd0-40e2-af79-2d532c390d09/Groot/newpassword/admin
